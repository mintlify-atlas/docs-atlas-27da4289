# Welcome to Cat Data API

Cat Data API is a REST API for managing cat-related images with secure authentication and PostgreSQL storage. Built with Express.js and TypeScript, it provides a robust backend for applications that need to store and retrieve cat images.

## Quick Overview

The Cat Data API provides four main endpoints:

- **POST /api/upload** - Upload a new cat image (JPG or PNG)
- **GET /api/images** - Retrieve all stored images
- **GET /api/image/:id** - Get a specific image by ID
- **DELETE /api/image/:id** - Delete an image

All endpoints require Auth0 JWT authentication for secure access.

## Technology Stack

- **Runtime**: Node.js with TypeScript
- **Framework**: Express.js
- **Database**: PostgreSQL with Knex.js
- **Authentication**: Auth0 with JWT
- **File Upload**: Multer
- **CORS**: Enabled for cross-origin requests
